Programs Client

class selfhost_client.programs_client.ProgramsClient(base_url: Optional[str] = None, username: Optional[str] = None, password: Optional[str] = None)

Bases: selfhost_client.base_client.BaseClient

A client for handling the program section of NODA Self-host API

__init__(base_url: Optional[str] = None, username: Optional[str] = None, password: Optional[str] = None) → None

BaseClient constructor

Sets the base url for the NODA Self-host API and authentication credentials in a new requests session. Will look for the following environment variables if parameters are omitted:

SELF_HOST_BASE_URL

SELF_HOST_USERNAME

SELF_HOST_PASSWORD

Parameters

  • base_url (Optional[str]) – Base url to the NODA Self-host API.

  • username (Optional[str]) – Username for the NODA Self-host API. The domain name is the username in this case.

  • password (Optional[str]) – Password for the NODA Self-host API. The access key is the password in this case.

Raises

SelfHostFatalErrorException – The client could not find auth credentials and a base url for the NODA Self-host API

create_program(name: str, program_type: str, state: Optional[str] = None, schedule: Optional[str] = None, deadline: Optional[int] = None, language: Optional[str] = None, tags: Optional[List[str]] = None) → selfhost_client.types.program_types.ProgramType

Add a new program to the NODA Self-host API

Parameters

  • name (str) – The name of the program

  • program_type (str) –

    • module: Modules are used by Routines and Webhooks to extend their functionality.

    • routine: Routines are executed at an interval.

    • webhook: Webhooks are called using the REST API.

  • state (Optional[str]) –

    The state of the program.

    • active

    • inactive

    • failed

  • schedule (Optional[str]) – A CRON schedule on the typical CRON format, yet with support for seconds. Ignored for Modules and Webhooks.

  • deadline (Optional[int]) – An duration (in milliseconds) after which a Program (routine, webhook) shall terminate, to avoid long running programs. Ignored for Modules.

  • language (Optional[str]) –

    • tengo

  • tags (Optional[List[str]]) – A list of tags pinned to the program

Returns

ProgramType

Raises

  • SelfHostBadRequestException – Sent request had insufficient data or invalid options.

  • SelfHostUnauthorizedException – Request was refused due to lacking authentication credentials.

  • SelfHostForbiddenException – Server understands the request but refuses to authorize it.

  • SelfHostTooManyRequestsException – Sent too many requests in a given amount of time.

  • SelfHostInternalServerException – Server encountered an unexpected condition that prevented it from fulfilling the request.
