Dataset Types

class selfhost_client.types.dataset_types.DatasetType(**kwargs)
uuid

The unique identifier for this dataset.

Type

str

name

Name of the resource. Does not have to be unique.

Type

str

format

File format of the data set.

Type

str

checksum

The sha256 checksum of the content.

Type

str

size

The size of the content in number of bytes.

Type

int

thing_uuid

A UUID reference to a Thing as a way to track data-sets to things.

Type

str

created

Date-time when created, as defined by RFC 3339, section 5.6.

Type

str

created_by

User UUID reference.

Type

str

updated

Date-time of last change, as defined by RFC 3339, section 5.6.

Type

str

updated_by

User UUID reference.

Type

str

tags

Tags pinned on this dataset.

Type

List[str]

    'uuid': '5e029cdf-4fee-42d2-9196-afbdfbdb9d8f'
