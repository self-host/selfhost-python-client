Program Types

class selfhost_client.types.program_types.ProgramType(**kwargs)
uuid

Unique identifier of the program.

Type

str

name

The name of the program.

Type

str

type

  • module: Modules are used by Routines and Webhooks to extend their functionality.

  • routine: Routines are executed at an interval.

  • webhook: Webhooks are called using the REST API.

Type

str

state

The state of the program.

  • active

  • inactive

  • failed

Type

str

schedule

A CRON schedule on the typical CRON format, yet with support for seconds. Ignored for Modules and Webhooks.

Type

str

deadline

An duration (in milliseconds) after which a Program (routine, webhook) shall terminate, to avoid long running programs. Ignored for Modules.

Type

int

language

  • tengo

Type

str

tags

A list of tags pinned to the program.

Type

List[str]

Example:

{
    'uuid': '47daa6eb-bd1c-49de-9782-1e9422a206f5'
    'name': 'My program',
    'type': 'routine',
    'state': 'active',
    'schedule': '0 45 23 * * 6',
    'deadline': 500,
    'language': 'string',
    'tags': [
        'tag1',
        'tag2'
    ],
}
